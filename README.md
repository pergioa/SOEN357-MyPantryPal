## Description
This is the minimal viable version of MyPantryPal.

## Features

- Home page introducing the pantry workflow
- Pantry item create, edit, delete, clear, and sample data actions
- Expiration date tracking with urgency badges
- Local storage persistence

## Added Compared To Previous Step

- This is the starting point.

## Run

```bash
npm install
npm start
```

Build with:

```bash
npm run build
```
